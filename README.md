## 代码规范

-   所有namespace（除sjtu）、类名、类的public函数名、非类成员函数名全部使用Pascal命名法。

    如：MyClass, MyClass::MyPublicFunction()

-   所有非类private变量使用驼峰命名法。

    如：myVariable

-   所有类的private成员（函数、变量）由单个下划线开头，后接驼峰命名法。

    如：_myPrivateFunction(), _myPrivateVariable
